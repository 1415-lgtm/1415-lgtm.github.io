第七步 部署到网站上

第一步在GitHub上创建一个仓库

命名 xxx.github.io

第二步 迁移到仓库地址

1

hugo --theme=LoveIt --baseUrl="[https://1415-lgtm.github.io/](http://1415-lgtm.github.io/)" -D

然后 进入创建好的public文件夹

1

cd public

使用git push到仓库

git init

git add .

git commit -m “first push”

git remote add origin https://github.com/1415-lgtm/1415-lgtm.github.io.git

git push -u origin master

后续更新

1

hugo --theme=LoveIt --baseUrl="[https://1415-lgtm.github.io/](http://1415-lgtm.github.io/)" -D

git add .

git commit -m “first push update”
